Press A Key to Start


How to play :

1) Press the START button/ any key on the key board. System will generate the first signal. Repeat the signal by pressing the same color button.
2) System will again generate signal. Repeat these two signals by pressing the same color buttons, in order.
3) System will duplicate these first two patten and add one.
4) Continue playing as long as you can repeat each sequence of signals correctly.
5) If you fail to repeat a sequence exactly, System responds with a "RAZZ" sound. This means you've lost, and the sequence of signals ends.
